# AAI-501: Introduction to AI and Machine Learning

This repository contains coursework, projects, and assignments for AAI-501 - Introduction to Artificial Intelligence and Machine Learning, part of the MS in Applied Artificial Intelligence program at the University of San Diego.

## Course Overview

AAI-501 provides a foundational introduction to artificial intelligence and machine learning concepts, techniques, and applications. The course covers core AI/ML algorithms, methodologies, and practical implementations.

## Repository Structure

```
intro_ai_ml_aai501/
├── data/           # Datasets used in assignments and projects
├── notebooks/      # Jupyter notebooks for experiments and analysis
└── README.md       # This file
```

## Prerequisites

- Python 3.8 or higher
- Jupyter Notebook or JupyterLab

## Getting Started

### Installation

1. Clone this repository:
```bash
git clone <repository-url>
cd intro_ai_ml_aai501
```

2. Create a virtual environment (recommended):
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install required dependencies:
```bash
pip install -r requirements.txt
```

### Usage

Navigate to the `notebooks/` directory and launch Jupyter:
```bash
jupyter notebook
```
